Sympedia GenFw

A flexible and easy to use framework for software artifact generation. GenFw is not a generator itself, rather it integrates existing input and output technologies, as well as several expression languages to tie the former together. Features include:

Content Providers Extensible via Eclipse ExtPoint. Implementations for DOM, Ecore, Properties.

Generators Extensible via Eclipse ExtPoint. Implementations for JET, JMerge, Java Formatter, Project Creation, DOM Transformation, ...

Rules Extensible via Eclipse ExtPoint. Implementations for BeanShell, OCL.

Modularization Generator configurations can be split into installable and reusable libraries.

User Interface Eclipse Actions and Views to manage and invoke generations. Extensible Editors for the configuration files.

Incremental Project Builder Automatically keep your generated code in sync with your models.

Ant Task Integrate your generator configurations into an Ant-based build process.

Density Generator A generator for reusable components, dependency injection containers and container integration tests. Runtime platform with container visualization.
