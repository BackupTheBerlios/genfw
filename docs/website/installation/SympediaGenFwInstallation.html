Abstract

Sympedia GenFw is a flexible and easy to use framework for software artifact generation. Input models can be associated with generators via extensible rule sets. Additional Content providers, generators and rules can be provided by Eclipse plugins.

This article explains the GenFw installation procedure.

Prerequisites

Be sure that you have the following software properly installed:

Eclipse 3.2 M4 I20060125-0800

EMF 2.2.0 I200601260027

GEF 3.2 200601260200 (only needed if you plan to use the Density Container Debugger)

The concrete versions are the ones I used when writing this.

Eclipse Update Manager

It is rather convenient to use the Eclipse Update Manager to install all the needed features and plugins. For this purpose Sympedia GenFw is available through an Eclipse Update Site.

Name: Sympedia GenFw Update Site

URL: http://genfw.berlios.de/update

Checkout the Sources

GenFw is available through its CVS respository at http://developer.berlios.de/cvs/?group_id=5851

    :pserver:anonymous@cvs.genfw.berlios.de:/cvsroot/genfw

Checkout the following modules from the util component:

    com.sympedia.util 
    com.sympedia.util.eclipse 
    com.sympedia.util.eclipse.resources 
    com.sympedia.util.eclipse.ui 
    com.sympedia.util.emf 
    com.sympedia.util.emf.edit 
    com.sympedia.util.emf.editor 
    com.sympedia.util-feature 
    org.apache.log4j 
    org.beanshell

Checkout the following modules from the genfw component:

    com.sympedia.genfw 
    com.sympedia.genfw.ant 
    com.sympedia.genfw.bsh 
    com.sympedia.genfw.bsh.edit 
    com.sympedia.genfw.edit 
    com.sympedia.genfw.editor 
    com.sympedia.genfw-feature 
    com.sympedia.genfw.jdt 
    com.sympedia.genfw.jdt.edit 
    com.sympedia.genfw.jet 
    com.sympedia.genfw.jet.edit 
    com.sympedia.genfw.jmerge 
    com.sympedia.genfw.jmerge.edit 
    com.sympedia.genfw.ui

To deploy GenFw into your Eclipse installation use the menu File | Export... | Deployable Features and select

    com.sympedia.util-feature
    com.sympedia.genfw-feature

If everything could be exported to the proper location without errors, you may now close the projects you checked out and restart Eclipse to load the installation.

Proceed with the Sympedia GenFw Tutorial 1.
